# Qlass Roadmap

## Alpha: one-teacher, one-class viability

- Reliable hosted deployment with HTTPS, Postgres, persistent uploads, and backups.
- CI coverage for Prisma validation, typecheck, tests, and production build.
- Google Classroom import dry-run and import logs.
- Alpha privacy notice, data export/delete workflow, and class-level alpha controls.
- Teacher-reviewed agentic unit builder with auditable runs.

## Self-hosted beta

- Checked-in Prisma migrations and upgrade guide.
- Docker Compose full-stack path with app, Postgres, and optional Redis.
- S3/Supabase storage guide and backup/restore documentation.
- Admin onboarding, instance health checks, and seed/demo mode.

## Agentic LMS differentiation

- Background agent job runner with resumable statuses.
- AI reviewer step for grade level, clarity, source safety, and standards alignment.
- Import-aware drafting using class history and course memory.
- Optional local-model recipe for Ollama, LM Studio, or vLLM.

## Connector and revenue opportunities

- Google Classroom import/export hardening.
- Grade CSV export and Moodle-compatible backup/export investigation.
- Schoology and Veracross feasibility research.
- Hosted Qlass teacher tier and managed small-school tier.
